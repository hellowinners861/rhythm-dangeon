# リズムダンジョン

日本一ソフトウェア『マッドラットデッド』を参考にした、**リズムアクション × 自動生成ダンジョン**のブラウザゲームです。
お気に入りの楽曲のビートに合わせてタイミングよくボタンを押すとキャラクターが行動し、自動生成された横視点ダンジョンを踏破してボスを倒すことを目指します。素のHTML/CSS/JS(Canvas)のみで作られており、ライブラリやビルドツールには依存していません。

## 遊び方

- 画面下部の ◀/▶ で前方へ1タイル移動、🦘 でジャンプ、⚔ で攻撃します。いずれも**曲のビートに合わせて押すと成立**し、ズレるとMISSになります。
- 判定はPERFECT/GOOD/MISSの3段階。判定成立の連続でコンボが伸び、一定数貯まると**フィーバー**(攻撃力アップ・コイン2倍)に突入します。
- 敵は予備動作(発光)の後に攻撃してくるので、音楽と見た目の両方に注意すれば避けられます。
- 全3章 × 各5ステージ構成で、各章の5ステージ目はボス戦です。ダンジョン中で拾ったコインで、出撃準備画面のショップから装備やキャラクターを購入できます。
- タイミングのズレが気になる場合は、オプションの「タイミング較正」でお使いの端末に合わせて補正できます。

## ローカルでの起動方法

ブラウザの `file://` 読み込みでは音源取得が制限されるため、簡易HTTPサーバー経由で開いてください。

```
python3 -m http.server 8000
```

起動後、ブラウザで `http://localhost:8000/` を開きます。

## 動作環境

**スマートフォンの横画面・タッチ操作専用**です。PCブラウザでも動作確認はできますが、縦画面では「横画面にしてください」の警告が表示されます。

## 注意事項

`songs/` 内の音源はユーザーの私用のためにリポジトリへ直接コミットしています。本リポジトリはpublicですが、GitHub Pages等での一般公開や、SNS等でのリポジトリURLの拡散はしないでください。
